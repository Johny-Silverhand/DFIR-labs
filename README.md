# DFIR & Security Research Lab

Привет! Это мой открытый учебный проект по цифровой криминалистике (DFIR) и анализу угроз.

Я учусь с нуля и документирую здесь весь процесс: лабораторные работы, конспекты, решения задач.

## Структура
*   **[Autopsy-Labs](./Autopsy-Labs/)** – Лабораторные работы по анализу образов дисков.
*   **[Memory-Forensics](./Memory-Forensics/)** – Разбор дампов оперативной памяти.
*   **[Guides](./Guides/)** – Полезные инструкции и шпаргалки по инструментам.

## Текущие цели
1.  Освоить базовый анализ в Autopsy.
2.  Научиться проводить расследование на учебных кейсах.
3.  Построить путь к профессии DFIR-специалиста.
